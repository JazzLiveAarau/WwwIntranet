Beschreibung

Das Fotoprogramm 'Snagit 9 Editor' wird für Plakate, Flyers und die Fotogalerien verwendet.
Snagit kann man im Internet kaufen, aber leider nicht Version 9. In der neuen Snagit Version (11) ist die für die Plakate wichtigste Funktion 'Color Correction' (sehe unten) weggenommen.


Installationsprogramm

Installationsprogramm 'Snagit.exe' ist auf dem Server im Ordner Installationsprogramme.



Der Webmaster hat das "Key".


Funktionen

Die folgende Funktionen werden normalerweise verwendet:

   - Grayscale
   - Color Effects -> Color Correction -> Contrast 50 - 100
   - Color Effects -> Color Depth ... -> Palette= fixed -> Colors 32, 16 oder 8
   - Color Effects -> Color Correction -> Brightness
   - Draw -> Rectangle -> Select -> Image -> Crop
   - Image -> Resize -> Resize Image -> Scale to specific size (pixels)
