Beschreibung

PDFCreator ist ein freies Programm für die Konvertierung von Word-Dokumente (und anderen Dokument) zu PDF-Dateien.


Installation

Das Programm PDFCreator ist "freeware". Ein Installationsprogramm kann man z.B. von sourceforge.net herunterladen.

Verwendung

1.
PDFCreator wird das Drucken von Flyern verwendet, d.h. ein PDF im richtigen Format und guter Auflösung wird kreiert.

2.
PDFCreator kann von jedem Programm mit einer Druckfunktion verwendet werden.
Beispiel: Word für die Kreierung von Plakaten.
