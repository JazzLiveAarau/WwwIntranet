Applikationen

Verwendung

Zwei Versionsverwaltungs-Systeme werden verwendet für das Verwalten von Quellcode:

-   GitHub für die Webapplikationen Homepage, Reservationen und Flyer.

-   TortoiseSVN für die Windowsapplikationen Admin, Newsletter und Adressen.

Der Quellcode wird in Repositories (Projektarchive) gespeichert.

Die Projektarchive der Webapplikationen sind auf der GitHub Homepage zugänglich, d.h. die können dort heruntergeladen werden.

Zwei Aufgaben erklären wie man mit GitHub arbeitet:
Aufgabe A0037 'Visual Studio Code für Homepage initialisieren'.

Aufgabe A0039 'VS Code für Web-Applikationen initialisieren'.
Das Projektarchiv (Repository) der Windowsapplikationen ist auf einem Memorystick.

Was ist GitHub?

Im Internet gibt es mehrere Videoanleitungen (Tutorials). Diese 'Git & GitHub Tutorial for Beginners' erklärt alles über Git und GitHub und Begriffe wie 'repository, stage, commit, push, pull, clone, ...' sehr gut.

Für JAZZ live AARAU wird nur ein Teil von allen Git Möglichkeiten verwendet. Im A0042 GitHub Beschreibung wird erklärt was wir verwenden.

Installation GitHub

Die Installation von GitHub ist in der Aufgabe A0037 'Visual Studio Code für Homepage initialisieren' beschrieben.

Installation TortoiseSVN
