Applikationen

Konzertsaison XML

Für jedes Konzertsaison gibt es eine XML Datei: JazzProgramm_2016_2017.xml, JazzProgramm_2016_2017.xml,…

Eine XML Datei ist wie eine Excel-Tabelle. Sie hält Daten wie „Name des Konzerts“, Datum, Kurzbeschreibung, etc.


Applikation XML

Die Texte für die grafische Benutzeroberfläche. „Labels“ wie Programm, Kontakt, Musiker Information, etc
sind in der Datei JazzApplication.xml definiert.

Diese Datei hält auch Daten über die Vorstandsmitglieder, die Spaghetti Factory, Starttexte für E-Mails, etc.

XML Ordner auf der Server

Die Konzertsaison XML und Applikation Dateien sind in dem XML Ordner auf der Server gespeichert.

Bei jedem Start von der App werden diese XML Dateien geholt. Die Dateien werden nie im Telefon gespeichert.
