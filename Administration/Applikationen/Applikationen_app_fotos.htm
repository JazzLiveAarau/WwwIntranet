Bilder vom Server

Die App holt alle gezeigte Bilder vom Server.


Plakate

Die Plakate sind die gleiche wie auf der Homepage und im Newsletter gezeigt werden.

Diese Bilder sind auf dem JAZZ live AARAU Server im Ordner "images".


Icons

Die Icons sind auf dem JAZZ live AARAU Server im Ordner "ImagesForApps".


Fotos der Vorstandsmitglieder

Die Fotos der Vorstandsmitglieder sind auf dem JAZZ live AARAU Server im Ordner "ImagesForApps/Members".


Konzertfotos

Die Konzertfotos sind die gleiche wie in den Fotogalerien und in der Slideshow gezeigt werden.

Die Konzertfotos sind auf dem JAZZ live AARAU Server in den Ordnern "FotoGalerie", "FotoKonzerte" und "FotoPublikum".
