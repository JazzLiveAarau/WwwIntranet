Beschreibung

Eine JAZZ live AARAU App für Android und iPhone Telefone und Tablets.

Die App ist mit Visual Studio und Xamarin entwickelt. Xamarin ist im Visual Studio integriert.

Visual Studio ist ein Windows Programm. Für die iPhone Version der App ist zugang zu einem Mac Computer mit
dem Programm Xcode auch notwendig.

Source Code ist auf dem JAZZ live AARAU Server im Ordner JazzAppSourceCode gespeichert.



App basiert auf XML Dateien

Daten, die bei der App gezeigt werden, kommen von XML Dateien der App.


Fotos vom Server

Fotos, Plakate und Icons sind auf dem JAZZ live AARAU Server gespeichert.



Neue Konzertsaison, Änderung von Texten und neue Fotos

Alle Änderungen werden wit der Windows-Applikation Admin gemacht

Einloggen

Als Vorstandsmitglied kann man sich einloggen. Damit wird in der App zusätzliche Information zugänglich wie zum Beispiel:

- Vollständige Kontaktdaten zu allen Mitgliedern
- Kontaktdaten für die Bands
- Das neue, noch nicht publizierte Saisonprogramm.
- Vorschläge für neue Funktionen für die App.

Die App ist für Android Telefone in Google Play Store und für iPhones in Apple App Store zugänglich.
Benutzername und Passwort zum Einloggen bekommt man vom Webmaster.
