Beschreibung

Das Excel Dateiformat CSV steht für englisch Comma-separated values (Wikipedia).

Die Dateinamenserweiterung lautet .csv.


Adressliste für Programm Adressen

Auf der JAZZ live AARAU Website gibt es eine Adressenliste im Excel (CSV) Format.
Trennzeichen für die Daten ist Komma.

Am Start des Programms Adressen wird die aktuelle Adressliste automatisch heruntergeladen.
Die Datei JazzLiveAdressen.csv ist im Ordner Excel.


Adressliste im Excel öffnen

Wenn man die Adressliste im Programm Excel anschauen möchte muss man:
1. Die Datei als Textdatei ins Excel importieren und andere Excel Funktionen verwenden (ziemlich kompliziert).
oder
2. Die Datei in Notepad oder Notepad++ öffnen und alle Kommas mit Semikolon ersetzen.
oder
3. Die Datei in Notepad oder Notepad++ öffnen und einer ersten Zeile „sep=,“ einfügen


Output CSV vom Programm Adressen

Die Output CSV Listen vom Programm Adressen haben Semikolon as Trennzeichen.

Diese CSV Dateien können direkt im Excel geöffnet werden.
