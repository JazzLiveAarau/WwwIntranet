Programm WS_FTP95 LE

Installation

Installationsprogramm ist auf dem Server im Ordner Installationsprogramme.




Einstellungen

Nach Start vom Programm:

   - New
   - JAZZ live AARAU, www.jazzliveaaerau.ch, jazzliv1, Password.
   - Startup, www, C:\Jazz\MyWeb
   - Options, no sounds



Herunterladen von Dateien

   - Datei in 'Remote Site' wählen z.B. JazzProgramm.htm.
   - Pfeile <- wählen.




Hochladen von Dateien

   - Datei in 'Local System' wählen z.B. JazzProgramm.htm.
   - Pfeile -> wählen.
