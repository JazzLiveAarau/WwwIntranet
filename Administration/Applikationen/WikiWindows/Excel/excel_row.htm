Row class

The class Row holds the data for a row in a table. Has functions to add, remove and retrieve data.

Field[] m_fields All fields in the row
int m_number_colums Number of columns (fields)

RemoveField(index_field, error) Remove field with a given index
InsertField(index_field, Field field, error) Insert a field where the position is defined by index
SetFieldValue(index_field, string value, error) Set field value for field defined by the index
Field GetField(index_field, error) Get field for a given index
