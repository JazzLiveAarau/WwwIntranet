Column class

Holds the data for a column in a table. Has functions to add and retrieve column data


Member variables

ArrayList m_array_fields List of fields
Field[] m_fields All fields in the column
int m_number_fields Number of fields in the column


Member functions

int NumberFields Get number of fields in the column
Column() Constructor
Clear() Clears the column, i.e. deletes all fields
AppendField(Field i_field) Appends a field to the column
Field GetField(index_field, error) Get field for a given index
