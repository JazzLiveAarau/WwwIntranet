FieldHeader class 2012-12-02

FieldHeader class

The class FieldHeader holds header data for a field (a column). Has functions to set and retrieve data


Member variables

string m_field_name Name of the field (column)
string m_field_caption Caption for the field (column)
FieldType m_field_type Type of field (column)
string m_field_help Help (tool tips) for the field (column)


Member functions

FieldHeader(field_name) Constructor that sets the name of the field (column)
string Name Get and set field (column) name
string Caption Get and set field (column) caption
FieldType Type Get and set field (column) type
string Help Get and set field (column) help/tooltips


Field type

The enumerator FieldType defines the type of the field. The following values are defined:
UNDEFINED STRING INTEGER FLOAT BOOLEAN
