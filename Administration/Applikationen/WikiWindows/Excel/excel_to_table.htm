ToTable class

The class ToTable has functions that convert "table" files in different formats to Table.

Implemented is the conversion of a .CSV file.

Other file types that could be implemnted are XSL, XSLX, XML.


Member variable

Allowed delimiters in the CSV file are semi-colon, colon and comma.
They are defined in a string array:

string[] m_csv_delimiters
