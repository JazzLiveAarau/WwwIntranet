FromTable class

The class ToTable has functions that convert a Table to "table" file in different formats.
Implemented is the conversion to a .CSV file and to .XSLX file.

Other file types that could be implemnted are XSL and XML.

For the conversion to XSLX files Excel must be installed in the computer.


Member variables

None

Member functions

TableToCsv(Table, file_name, delimiter, Encoding, error) Convert a table to a .csv file
TableToXlsx(Table, file_name, error) Convert a table to a .xlsx file
