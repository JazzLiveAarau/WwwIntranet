Installation

Das Programm XML Notepad ist "freeware". Ein Installationsprogramm kann man von Microsoft-Download herunterladen.

Syntax Kontrolle mit XSD Datei

Beim Änderungen von XML Dateien mit Programm XML Notepad sollte man eine Syntax-Kontrolle mit einer XSD Datei machen
Die XSD Datei definiert erlaubte XML elemente und wo sie im "XML-Baum" (XML-Tree) erlaubt sind.

Mit View -> Schemas wird eine XSD Datei definiert:




XSD Datei generieren

Eine XSD Datei kann von einer XML Datei generiert werden. In Visual Studio gibt es eine Funktion dafür.

Wenn diese Funktion verwendet wird, muss man natürlich sicher sein, dass die XML Datei fehlerfrei ist.
