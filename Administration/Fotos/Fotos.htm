Fotos

Fotos von Konzerten

Fotos von den Musikern werden bei fast jedem Konzert gemacht.

Die Fotos werden auf die Homepage gestellt und auf dem Server gespeichert.

Eine Fotogalerie besteht aus neun Fotos von einem Konzert. Für viele Konzerte gibt es zwei Fotogalerien auf der Homepage.

Auf der Homepage kann man die Fotos als Fotoshow anschauen.

Homepage Besucher können Original-Fotos (hohe Auflösung) verlangen.


Server-Ordner und Dateien der Fotos

Mit einem FTP Programm kann Foto-Dateien angeschaut oder heruntergeladen werden. Die URLs (Server Ordner) sind:
- Fotogalerie 1: /www/FotoGalerie/
- Fotogalerie 2: /www/FotoKonzerte/
- Originalbilder, komprimiert zu ZIP Dateien: /images/
- XML Dateien für die Galerien, Publikum und Gästebuch: /www/XML/
