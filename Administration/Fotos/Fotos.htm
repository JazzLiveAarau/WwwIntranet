Fotos von Konzerten

Fotos von den Musikern werden bei fast jedem Konzert gemacht.

Die Fotos werden auf die Homepage gestellt und auf dem Server gespeichert.

Eine Fotogalerie besteht aus neun Fotos von einem Konzert. Für viele Konzerte gibt es zwei Fotogalerien auf der Homepage.

Auf der Homepage kann man die Fotos als Fotoshow anschauen.

Homepage und App Besucher können Original-Fotos (hohe Auflösung) verlangen.


Server-Ordner und Dateien der Fotos

Die Fotogalerie 1 ist auf dem Server im Ordner /www/FotoGalerie/.

Fotogalerie 2 ist auf dem Server im Ordner /www/FotoKonzerte/.

Alle Originalbilder, komprimiert zu ZIP Dateien, sind auf dem Server im Ordner Konzertfotos.
Es ist eine ZIP-Datei pro Konzert und Galerie (Fotograf).

Die XML Dateien JazzFotoGalerieEin.xml und JazzFotoGalerieZwei.xml registrieren Fotogalerien und ZIP-Dateien.
Diese XML Dateien sind auf dem Server im Ordner /www/XML/.
