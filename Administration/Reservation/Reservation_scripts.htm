Reservationen

Wie ist die Web-Applikation implementiert?

Die Applikation ist basierend auf HTML, JavaScript, XML, SVG und PHP.


HTML

HTML ist eine Sprache für das Kreieren von Webseiten.


JavaScript

JavaScript ist eine Sprache für das Programmieren von Webseiten.

Die JavaScript Funktionen werden im (eigenen) Computer ausgeführt


XML

XML ist einer Excel-Tabelle ähnlich.

Die XML Dateien für diese Applikation enthalten Daten für das Lokal und die Reservationen. Beispiele:

Tishplandaten
Reservationsdaten


SVG

SVG = Scalable Vector Graphics.

Bilder mit SVG sind gut skalierbar, d.h. sie sind immer scharf auch wenn man sie
auf dem Schirm maximal vergrössert werden.


PHP

PHP ist eine Sprache für das Programmieren von Webseiten sowie Kreieren und Ändern von Dateien auf dem Server.
Für diese Reservations-Applikation geht es um die XML Dateien für den Tischplan und die Reservationen.

Die PHP Funktionen werden auf dem Server (Computer) ausgeführt
