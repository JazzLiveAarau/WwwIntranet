Hostpoint

Hostpoint ist unser Internet Provider.

Die Homepage und die anderen Web-Applikationen werden mit Visual Studio Code entwickelt und der (Original-) Quellcode wird auf der GitHub Website gespeichert.

Mit einem FTP Programm werden den Quellcode der Applikationen auch auf dem Hostpoint Server hochgeladen.
Dieser QuellCode ist was verwendet wird, aber man soll den Code als Kopie betrachten. Original-Code ist im GitHub.

Konzertdaten, Fotos und andere Daten werden mit der Windows-Applikation Admin zum Hostpoint Server aufgeladen.
Auch ein paar von den Web-Applikationen laden Daten zum Hostpoint Server hoch.

Einige wenige Daten werden auch "manuell" mit einem FTP Programm hochgeladen.

Die E-Mail Konten und Weiterleitungen kriegen wir auch von Hostpoint.

Vertragsinhaber

Ein Vorstandsmitglied ist der Inhaber des Servers.

Wie man Vertragsinhaber ändert ist hier beschrieben.

Rechnung

Einmal pro Jahr schickt Hostpoint eine Rechnung.

Die Rechnungsadresse (Person) kann geändert werden:
- Login Hostpoint mit Hostpoint ID - Menu ADMIN wählen - Menu Rechnung-Adresse wählen

Rechnungen können auch nach Login bezahlt werden.

Passwort

Passwort für Hostpoint ist vom Webmaster erhältlich.

Für das einrichten von einem FTP Programm ist das Passwort notwendig.

Login Website

Nur mit einer persöhnlichen Hostpoint ID, kann man sich einloggen.
In Aufgabe A0032 ist beschrieben wie man eine Hostpoint ID bekommt.
